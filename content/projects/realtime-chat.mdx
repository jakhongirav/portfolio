A feature-rich chat platform that provides secure, real-time communication capabilities. This project demonstrates my expertise in building real-time applications with complex state management and security requirements.

## Key Features

- Real-time messaging with WebSocket
- End-to-end encryption
- File sharing and media messages
- Video and audio calls
- Group chat support
- Message search and filtering
- Offline support with message queuing
- Read receipts and typing indicators

## Technical Details

- **Frontend**: React with TypeScript
- **Backend**: Node.js with Socket.io
- **Database**: MongoDB for message persistence
- **File Storage**: AWS S3
- **Video Calls**: WebRTC
- **State Management**: Redux Toolkit
- **UI Framework**: Chakra UI
- **Testing**: Cypress for E2E testing

## Security Features

- End-to-end encryption using Signal Protocol
- Secure file transfer
- Two-factor authentication
- Message expiration
- Device management
- Audit logging

This project showcases my ability to build complex real-time applications while maintaining high security standards and excellent user experience.
