My personal portfolio website showcases my work and skills as a software developer. Built with modern web technologies, it demonstrates my expertise in front-end development and UI/UX design.

## Tech Stack

- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Content**: MDX with Contentlayer
- **Animations**: Framer Motion
- **Deployment**: Vercel

## Key Features

- Responsive design that works on all devices
- Dark mode support
- Server-side rendering for optimal performance
- Dynamic content management using MDX
- Beautiful animations and transitions
- SEO optimized
- Analytics integration
